## 一、最高人民法院官方样式（管辖权异议上诉）

### 标准模板

```
民事上诉状

上诉人(原审诉讼地位)：×××，男/女，××××年××月××日出生，×族，……(写明工作单位和职务或者职业)，住……。联系方式：……。

法定代理人/指定代理人：×××，……。

委托诉讼代理人：×××，……。

被上诉人(原审诉讼地位)：×××，……

(以上写明当事人和其他诉讼参加人的姓名或者名称等基本信息)

上诉人×××因与被上诉人×××……(写明案由)一案，不服××××人民法院××××年××月××日作出的(××××)……民初……号驳回管辖权异议裁定，现提起上诉。

上诉请求：

1. 撤销××××人民法院(××××)……民初……号驳回管辖权异议民事裁定书；
2. 本案移送××××人民法院处理。

上诉理由：

……(写明不服驳回管辖权异议裁定的事实和理由)。

此致

××××人民法院

附：本上诉状副本×份

上诉人(签名或者盖章)

××××年××月××日
```

## 二、通用民事上诉状模板（实体判决上诉）

### 扩展模板

```
民事上诉状

上诉人(原审原告/被告)：×××，男/女，××××年××月××日出生，×族，工作单位：×××，职务：×××，住×××。联系方式：×××。

法定代理人/指定代理人：×××，……。

委托诉讼代理人：×××，……。

被上诉人(原审被告/原告)：×××，男/女，××××年××月××日出生，×族，工作单位：×××，职务：×××，住×××。联系方式：×××。

(以上写明当事人和其他诉讼参加人的姓名或者名称等基本信息)

上诉人×××因与被上诉人×××……(写明案由)一案，不服××××人民法院于××××年××月××日作出的(××××)……民初……号民事判决/裁定，现依法提起上诉。

上诉请求：

1. 请求撤销××××人民法院作出的(××××)……民初……号民事判决/裁定；
2. 请求依法改判_________________________(具体改判内容)；
   或请求将本案发回××××人民法院重审；
3. 本案一、二审诉讼费用由被上诉人承担。

上诉理由：

上诉人认为，一审判决认定事实不清/适用法律错误/程序违法，具体理由如下：

一、认定事实不清

(详细陈述一审判决在认定事实方面存在的问题)
例如：一审法院在认定……方面存在事实错误，具体表现为……
证据显示：……(列举具体证据)
但一审判决未予采信/未予查明，导致事实认定错误。

二、适用法律错误

(详细陈述一审判决在适用法律方面存在的错误)
例如：一审法院错误适用了《××法》第××条的规定，本案应当适用《××法》第××条。
根据……(法律规定)，本案应当认定为……，但一审判决认定为……，显属适用法律错误。

三、程序违法(如有)

(如有程序违法问题，在此说明)
例如：一审法院未依法送达诉讼文书/未依法组织质证/未保障当事人的辩论权利等。

综上所述，一审判决存在严重错误，恳请二审法院依法查明事实，正确适用法律，支持上诉人的上诉请求，以维护上诉人的合法权益。

此致

××××中级人民法院

附：本上诉状副本×份

上诉人(签名或者盖章)：

××××年××月××日
```

## 三、法言法语特征分析

### 1. 上诉请求的表述方式

#### 标准三段式请求：
1. **撤销原判**："请求撤销××××人民法院作出的(××××)……民初……号民事判决/裁定"
2. **改判或发回**：
   - 改判："请求依法改判_________________________"
   - 发回："请求将本案发回××××人民法院重审"
3. **费用承担**："本案一、二审诉讼费用由被上诉人承担"

#### 用词特点：
- **"不服"**：上诉状专用词，表达对原判的不接受态度
- **"依法提起上诉"**：表明上诉行为的合法性
- **"依法改判"**：请求二审法院重新作出判决
- **"发回重审"**：请求将案件退回一审法院重新审理

### 2. 事实与理由部分的论述方式

#### 段落结构（"一、二、三"序号）：

**第一部分：认定事实不清**
- **标准表述**："一审判决认定事实不清"
- **具体展开**："一审法院在认定……方面存在事实错误"
- **证据列举**："证据显示：……(列举具体证据)"
- **错误指出**："但一审判决未予采信/未予查明，导致事实认定错误"

**第二部分：适用法律错误**
- **标准表述**："一审判决适用法律错误"
- **法条对比**："一审法院错误适用了《××法》第××条的规定，本案应当适用《××法》第××条"
- **法理分析**："根据……(法律规定)，本案应当认定为……，但一审判决认定为……，显属适用法律错误"

**第三部分：程序违法（如有）**
- **标准表述**："一审法院程序违法"
- **违法情形**："未依法送达诉讼文书/未依法组织质证/未保障当事人的辩论权利等"

#### 句式特征：
- **否定性评价**："认定事实不清"、"适用法律错误"、"程序违法"
- **对比论证**："应当……但……"的转折句式
- **结论性断语**："显属……错误"、"严重违反法定程序"

### 3. 法律依据的引用方式

#### 上诉状法条引用特点：
- **对比式引用**：同时指出"错误适用的法条"和"应当适用的法条"
- **规范性表述**："根据《××法》第××条之规定"
- **适用结论**："本案应当认定为……"

#### 引用语式：
- "一审法院错误适用了……规定"
- "本案应当适用……"
- "根据……(法律/司法解释)"
- "显属适用法律错误"

### 4. 结尾格式

#### 总结段结构：
1. **错误归纳**："综上所述，一审判决存在严重错误"
2. **二审请求**："恳请二审法院依法查明事实，正确适用法律，支持上诉人的上诉请求"
3. **价值升华**："以维护上诉人的合法权益"

#### 落款格式：
```
此致
××××中级人民法院

附：本上诉状副本×份

上诉人(签名或者盖章)：

××××年××月××日
```

#### 格式要点：
- **"此致"**：单独一行，后跟二审法院名称
- **二审法院**：中级人民法院（一般案件）或高级人民法院/最高人民法院（特殊案件）
- **"附"**：注明上诉状副本份数（按被上诉人人数提交）
- **上诉人签名**：自然人亲笔签名，法人或其他组织加盖公章

## 四、上诉的法律依据与期限

### 1. 上诉期限规定

根据《中华人民共和国民事诉讼法》：
- **判决上诉**：判决书送达之日起**15日内**
- **裁定上诉**：裁定书送达之日起**10日内**（管辖权异议裁定）
- **涉外上诉**：在领域内无住所的当事人，上诉期为**30日**

### 2. 上诉状的法律依据

根据《中华人民共和国民事诉讼法》相关条文：
- **第一百五十四条**：对裁定上诉的程序规定
- **第一百六十四条**：上诉期限规定
- **第一百六十五条**：上诉状应当包括的内容
- **第一百六十六条**：上诉状提交方式

### 3. 上诉状提交方式

- **应当通过原审人民法院提出**
- **按照对方当事人或者代表人的人数提出副本**
- **原审法院收到后在5日内送达对方当事人**

## 五、上诉状与起诉状/答辩状的区别

### 1. 称谓区别
| 文书类型 | 当事人称谓 |
|---------|-----------|
| 起诉状 | 原告、被告 |
| 答辩状 | 答辩人（被告）、被答辩人（原告） |
| 上诉状 | 上诉人（原审原告/被告）、被上诉人（原审被告/原告） |

### 2. 请求区别
- **起诉状**：提出诉讼请求，要求法院支持
- **答辩状**：反驳诉讼请求，要求法院驳回
- **上诉状**：不服原判，要求撤销/改判/发回

### 3. 论述重点区别
- **起诉状**：原告的请求权基础+事实理由
- **答辩状**：针对原告诉讼请求的反驳
- **上诉状**：针对一审判决错误的分析

## 六、常见上诉理由类型

### 1. 认定事实不清
- 关键事实认定错误
- 证据采信不当
- 事实遗漏或曲解

### 2. 适用法律错误
- 引用法条错误
- 法律关系定性错误
- 法律责任划分错误

### 3. 程序违法
- 未依法送达
- 未保障辩论权利
- 质证程序违法
- 审判组织不合法

### 4. 裁判显失公平
- 自由裁量权滥用
- 赔偿数额畸高/畸低
- 责任划分明显不当

本文件提供了两个版本的民事上诉状模板：
1. **管辖权异议上诉专用模板**（最高院官方样式）
2. **通用实体判决上诉模板**（基于上诉实务扩展）

使用时请根据案件实际情况选择合适的模板，并注意：

- 严格在法定期限内提起上诉

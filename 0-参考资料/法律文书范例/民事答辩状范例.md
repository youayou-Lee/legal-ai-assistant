## 一、最高人民法院官方样式（法人或其他组织）

### 标准模板

```
民事答辩状

答辩人：×××，住所地……。

法定代表人/主要负责人：×××，……(写明职务)，联系方式：……。

委托诉讼代理人：×××，……。

(以上写明答辩人和其他诉讼参加人的姓名或者名称等基本信息)

对××××人民法院(××××)……民初……号……(写明当事人和案由)一案的起诉，答辩如下：

……(写明答辩意见)。

证据和证据来源，证人姓名和住所：

……

此致

××××人民法院

附：本答辩状副本×份

答辩人(公章和签名)

××××年××月××日
```

### 样式说明

1. 本样式根据《中华人民共和国民事诉讼法》第一百二十五条制定，供法人或者其他组织对民事起诉提出答辩用。
2. 被告应当在收到起诉状副本之日起**十五日内**提出答辩状。被告在中华人民共和国领域内没有住所的，应当在收到起诉状副本后**三十日内**提出答辩状。
3. 答辩时已经委托诉讼代理人的，应当写明委托诉讼代理人基本信息。
4. 答辩状应当加盖单位印章，并由法定代表人或者主要负责人签名。

## 二、自然人答辩状详细范本（装饰装修合同纠纷示例）

### 完整范例

```
民事答辩状

答辩人（本诉被告）：
公司名称：xxxx，统一社会信用代码：xxxx，住址：xxxxxxxxxxxxxxxxxxx。

法定代表人：xxxx，联系电话：xxxxxxxx。

被答辩人（本诉原告）：
姓名：xxx，性别：x，xxxx年x月xx日出生，身份证号：xxxxxxxxxxxxxxxx，住址：xxxxxxxxxxxxx，联系电话：xxxxxxxxxxxx。

答辩人（本诉被告xxxx，以下简称"被告"）与被答辩人（本诉原告xxx，以下简称"原告"）装饰装修合同纠纷一案，被告于2022年8月5日收到贵院送达的民事起诉状副本。根据相关事实及法律规定，结合现有证据及原告提交的民事起诉状，被告认为原告在起诉状中所阐述的事实与真实情况明显不符，被告认可原告解除《装饰装修工程施工合同》的请求，但对原告提出的其他诉讼请求不予认可，恳请贵院依法驳回原告的其他诉讼请求。

事实及理由如下：

一、原告与被告于2021年11月25日签订《装饰装修工程施工合同》，约定由被告承揽原告室内的柜体定制工作及室内门改造工作（具体承揽工作详见报价表）。根据施工流程，被告所承揽的工作只能在室内所有基础装修施工完成后，方能进行施工。但由于原告与第三人合作的基础装修设施一直未能完工，导致被告的施工条件无法达成，因而始终无法进场施工。期间，被告多次与原告就此事进行沟通，但直到原告起诉之日起，都毫无进展，导致被告施工一直无法如期进行（有微信群聊天记录可以证明）。因此，造成工期延误的原因不在被告，而在原告。

根据《民法典》第七百七十八条之规定，承揽工作需要定作人协助的，定作人有协助的义务。定作人不履行协助义务致使承揽工作不能完成的，承揽人可以催告定作人在合理期限内履行义务，并可以顺延履行期限；定作人逾期不履行的，承揽人可以解除合同。被告所承揽的工作若需完成必须依赖于原告所提供施工场地及施工条件，但由于原告的原因，导致被告的施工条件一直无法达成，原告未履行协助义务造成被告未能按时完工，则履行期限可以顺延，且由此造成的责任应当由原告方承担。

二、原告支付第一期工程款65000元后，被告便按照与原告的约定开始准备施工所需工作，具体包括设计方案的准备、定制材料的制作与加工，并联系施工工人进行相关施工工作，其中已支出的费用包括设计费17000元（设计图纸费用、设计师去工地现场服务费用）、木门拆运费用1490元、木门工厂改造费用3660元，以及定制柜双饰面板材费用、运输费用、着色板材费用等加工费用25676元，共计47826元（详见加工清单）。

根据《民法典》第七百八十七条之规定，定作人在承揽人完成工作前可以随时解除合同，造成承揽人损失的，应当赔偿损失。被告认为，原告可以解除合同，但被告为履行合同义务已经支出费用47826元，该费用为被告的直接损失，应当由原告承担，被告的损失应当从原告已支付工程款中予以扣除，因此原告要求返还第一期全部工程款65000元，被告不予认可。

三、2021年12月22日至2022年1月23日，因爆发新冠疫情，政府实施了疫情封控政策，全城居家隔离，无法正常施工，该情况属于不能预见、不能避免并不能克服的不可抗力事件。根据《民法典》第一百八十条之规定，因不可抗力不能履行民事义务的，不承担民事责任。且上述已经阐明，工期延误的根本原因是由原告造成的。因此，原告请求"判令被告支付违约金4700元"的理由不成立。

四、关于室内门，原室内门为房屋自带，原告要求室内门与室内定制柜体颜色一致，但原告认为制作新的木门费用比较高，因此根据原告的要求，为了降低费用，由被告将室内所有门板拆除运送到工厂重新做颜色。原告现要求被告返还室内门并安装至正常使用，被告可以配合，但原告应当承担由此产生的费用。

综上，被告的上述答辩意见观点鲜明，理由充分，合法合理，应予支持。而原告的诉讼请求毫无事实和法律依据，严重损害了被告的合法权益。故，恳请贵院在查明事实的基础上，立即驳回原告的无理诉讼请求，让答辩人在本案中感受到司法的公平正义！

此致

xxxx人民法院

答辩人：
法定代表人：

年     月     日
```

## 三、法言法语特征分析

### 1. 答辩状的开场表述方式

#### 标准开场白结构：
- **收案说明**："答辩人于×年×月×日收到贵院送达的民事起诉状副本"
- **法律依据前置**："根据相关事实及法律规定，结合现有证据及原告提交的民事起诉状"
- **立场表明**："被告认为原告在起诉状中所阐述的事实与真实情况明显不符"
- **部分认可策略**："被告认可……，但对……不予认可"
- **最终请求**："恳请贵院依法驳回原告的诉讼请求"

#### 用词特点：
- "贵院"：对受理法院的尊称
- "被答辩人"：对起诉人的正式称呼（答辩状专用）
- "真实情况明显不符"：否定对方事实陈述的标准表述
- "恳请"：表示谦抑的请求用词

### 2. 事实与理由部分的论述方式

#### 段落结构（采用"一、二、三、四"序号）：

**第一段：事实重构**
- 先陈述客观事实（合同签订、约定内容）
- 指出对方过错（"造成工期延误的原因不在被告，而在原告"）
- 引用证据支持（"有微信群聊天记录可以证明"）

**第二段：法律适用**
- 引用具体法条（"根据《民法典》第七百七十八条之规定"）
- 结合案件事实（"被告所承揽的工作若需完成必须依赖于……"）
- 得出法律结论（"由此造成的责任应当由原告方承担"）

**第三段：损失计算**
- 详细列明费用（设计费、拆运费、改造费等）
- 总计损失金额（"共计47826元"）
- 法律依据支撑（"根据《民法典》第七百八十七条之规定"）

**第四段：免责事由**
- 陈述不可抗力（"疫情封控政策，全城居家隔离"）
- 引用法条免责（"因不可抗力不能履行民事义务的，不承担民事责任"）

#### 句式特征：
- **法条引用格式**："根据《××法》第××条之规定，……"
- **三段论推理**：大前提（法律规定）→小前提（案件事实）→结论（法律效果）
- **因果关系连接词**："由于"、"因而"、"导致"、"因此"

### 3. 法律依据的引用方式

#### 显式引用（答辩状特点）：
- **完整法条表述**："根据《民法典》第七百七十八条之规定"
- **关键词保留**："定作人有协助的义务"、"可以顺延履行期限"
- **法条功能说明**：每个法条引用后都紧跟案件适用说明

#### 引用语式：
- "……之规定"：法条引用的标准后缀
- "属于……事件"：定性判断的标准表述
- "被告认为，……"：观点陈述的引入语

### 4. 结尾格式

#### 总结段结构：
1. **自我评价**："被告的上述答辩意见观点鲜明，理由充分，合法合理，应予支持"
2. **对方评价**："而原告的诉讼请求毫无事实和法律依据，严重损害了被告的合法权益"
3. **最终请求**："故，恳请贵院在查明事实的基础上，立即驳回原告的无理诉讼请求，让答辩人在本案中感受到司法的公平正义"

#### 落款格式：
```
此致
xxxx人民法院

答辩人：
法定代表人：

年     月     日
```

## 四、答辩状的常见反驳方向

### 1. 事实不符类
- "原告起诉状中所述事实与实际不符"
- 示例：借款不存在、款项系还款、原告系过错方等

### 2. 法律依据缺失类
- "原告诉请没有法律依据"
- 示例：已偿还欠款、付款条件未成就、于法无据等

### 3. 逐项反驳类
- 根据民事起诉状列明的诉请事项逐项反驳
- 每个诉讼请求单独回应

## 五、答辩期限规定

### 根据《民事诉讼法》第125条：
1. **一审程序**：被告应当在收到起诉状副本之日起**15日内**提出答辩状
2. **二审程序**：原审法院收到上诉状后**5日内**送达被上诉人，被上诉人收到之日起**15日内**提出答辩状
3. **涉外案件**：在领域内无住所的，答辩期为收到起诉状副本后**30日**

## 六、答辩状撰写要点

### 核心原则：
1. **突出重点，不要重复**
2. **发挥防御和反击的作用**
3. **围绕对方诉讼请求进行反驳**
4. **给出所依据的事实和证据**

### 逻辑要求：
- 闭合、周严的逻辑
- 三段论的严谨推理
- 审慎斟酌的法言法语

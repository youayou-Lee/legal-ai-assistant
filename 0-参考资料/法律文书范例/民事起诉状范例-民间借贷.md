## 标准模板

```
                  民 事 起 诉 状

原告：xx，男/女，x族，xx年xx月xx日出生，现住xx市xx区xx路xx小区xx幢xx室，公民身份号码xxxxxx，联系电话：xxxxx。

被告一（借款人）：xx，男/女，x族，xx年xx月xx日出生，现住xx市xx区xx路xx小区xx幢xx室，公民身份号码xxxxxx，联系电话：xxxxx。

被告二（若有保证人）：xx，男/女，x族，xx年xx月xx日出生，现住xx市xx区xx路xx小区xx幢xx室，公民身份号码xxxxxx，联系电话：xxxxx。

诉讼请求：

1. 请求判令被告一归还借款xx元，并支付借款利息（按月利息x%的标准，自xx年xx月xx日起计算至实际履行之日止）【借条中约定利息的情况】；
   或判令被告一归还借款xx元，并支付借款利息（按中国人民银行同期LPR的标准，自xx年xx月xx日起计算至实际履行之日止）【借条中未约定利息的情况】。

2. 请求判令被告二承担保证责任（连带/一般）；

3. 本案的诉讼费用由被告一、二负担。

事实和理由：

原告和被告一系朋友关系，被告一因资金周转需要，于xx年xx月xx日向原告借款xx元。被告一出具借条一份，载明借款金额、借款时间、借款利率等。原告通过银行转账/微信转账/现金的方式将款项交付被告一，被告一仅支付xx个月的借款利息后，未继续支付利息，到期后也未归还借款本金，被告二也未履行保证义务。为此，原告诉至法院，请求判如所请。

此致
           人民法院

                                         具状人：xx（本人亲笔签名）
     年     月     日
```

## 法言法语特征分析

### 1. 诉讼请求的表述方式

#### 请求句式特征：
- **标准开头**："请求判令被告一……"
- **核心动词**：判令、承担、负担
- **利息表述**：
  - 有约定利息："按月利息x%的标准，自xx年xx月xx日起计算至实际履行之日止"
  - 无约定利息："按中国人民银行同期LPR的标准"
- **费用承担**："本案的诉讼费用由被告一、二负担"

#### 用词特点：
- "判令"：法院对当事人的命令性要求
- "连带/一般"：保证责任的两种法定类型
- "实际履行之日"：精确的时间节点表述

### 2. 事实与理由部分的论述方式

#### 叙述结构：
1. **关系说明**："原告和被告一系朋友关系"
2. **借款原因**："被告一因资金周转需要"
3. **借款时间**："于xx年xx月xx日向原告借款xx元"
4. **证据引述**："被告一出具借条一份，载明借款金额、借款时间、借款利率等"
5. **交付方式**："原告通过银行转账/微信转账/现金的方式将款项交付被告一"
6. **违约事实**："被告一仅支付xx个月的借款利息后，未继续支付利息，到期后也未归还借款本金"
7. **保证人违约**："被告二也未履行保证义务"
8. **诉讼引语**："为此，原告诉至法院，请求判如所请"

#### 句式特征：
- **时态使用**：全部使用过去时态（"向原告借款"、"出具借条"、"将款项交付"）
- **事实陈述客观化**：避免主观评价，只陈述客观事实
- **因果关系明确**：先陈述违约事实，再引出诉讼请求

### 3. 法律依据的引用方式

在民间借贷纠纷中，本模板采用**隐式引用**方式：
- 不直接引用法律条文编号
- 通过"借条中约定利息"、"保证责任"等表述隐含法律依据
- 重点在于陈述事实，让法院自行适用法律

### 4. 结尾格式

#### 标准三段式结构：
```
此致
           人民法院
                                         具状人：xx（本人亲笔签名）
     年     月     日
```

#### 格式要点：
- **"此致"**：单独一行，左对齐或居中
- **受诉法院**：另起一行，顶格或居中书写
- **具状人签名**：右对齐，注明"本人亲笔签名"
- **日期**：签名下方，中文大写或阿拉伯数字均可

## 法律依据说明（供参考）

### 起诉条件
根据《中华人民共和国民事诉讼法》第一百二十二条规定，起诉必须符合下列条件：
1. 原告是与本案有直接利害关系的公民、法人和其他组织
2. 有明确的被告
3. 有具体的诉讼请求和事实、理由
4. 属于人民法院受理民事诉讼的范围和受诉人民法院管辖

### 起诉状应载明事项
1. **标题**：单列一行在正中写"民事起诉状"或"民事诉状"
2. **诉讼参与人身份等基本情况**：姓名、性别、民族、出生日期、住址、身份证号、联系方式等
3. **诉讼请求**：明确、具体、简明扼要
4. **事实和理由**：先写事实，后写理由，是诉状的正文和核心
5. **落款**：受诉法院、具状人、日期

---

## 证据清单要点（需附在起诉状后）

1. 借条或借款合同原件
2. 银行转账记录/微信转账记录/收条
3. 被告身份信息证明
4. 利息计算明细（如有利息约定）
5. 保证合同或担保协议（如有保证人）

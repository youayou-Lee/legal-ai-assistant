# 2026-04-06 工作记录

## 今日完成

### 1. 法院文书自动处理 Skill（court-document-handler）✅

**问题背景**：律师经常收到法院送达短信（如广西法院短信平台），包含文书下载链接。律师可能忘记处理或错过重要时限。

**示例短信**：
```
【广西法院短信平台】闭潇月：南宁市青秀区人民法院向您发送了（2026）桂0103民初3262号案件相关文书，文书详见:http://171.106.48.55:28083/sfsdw//r/YhXWadrTF8
```

**解决方案**：当律师将法院短信复制发送给 OpenClaw 时，自动触发 court-document-handler Skill，全自动处理。

**完整流程**：
```
收到短信 → 提取链接 → 浏览器访问 → 手机验证(886288) → 获取文书 → 下载PDF → 提取信息 → 微信汇报+发送PDF → 设置提醒
```

**核心功能**：
1. 自动识别法院短信并触发
2. 用 OpenClaw browser 命令访问链接（法院系统需验证码，不能用 web_fetch）
3. 输入手机后六位验证码，获取文书
4. 下载 PDF 并提取案件关键信息（案号、案由、法院、当事人等）
5. 通过微信发送结构化汇报 + PDF 原件
6. 根据文书类型自动设置提醒（传票→开庭前1天，判决书→上诉截止前，举证通知→截止前3天）

**修复问题**：
- SKILL.md frontmatter 缺少闭合 `---`，导致 YAML 解析失败，OpenClaw 静默跳过不加载
- 修复后 `openclaw skills list` 正常显示 `✓ ready ⚖️ court-document-handler`

**部署位置**：`~/.agents/skills/court-document-handler/`

**对接渠道**：微信（通过 openclaw-weixin channel）

**⚠️ 限制**：
- 目前仅支持广西法院短信平台链接
- 验证码为固定手机后六位
- 待真实法院短信验证完整流程

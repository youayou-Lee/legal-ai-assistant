## 一、继承纠纷 — 标准答案

### 正确推理步骤（必须按此顺序）

**第一步：确认遗产范围（最关键，最容易跳步）**

1. **甲市房屋**：虽登记在王某名下，但系**婚姻关系存续期间**共同购买，属夫妻共同财产。离婚协议约定房屋归王某、王某补偿李某100万元，但王某**未实际支付补偿款**。
   - 关键判断：离婚协议是否已生效？→ 协议经双方签字即生效，房屋产权已依协议确定归王某所有
   - 但王某欠李某100万元补偿款构成**债务**，李某是王某的债权人，不是房屋共有人
   - 结论：房屋属王某个人财产（依离婚协议），应纳入遗产范围

2. **存款30万元**：需确认是婚前还是婚后取得。题目未特别说明婚后取得，推定为王某个人财产 → 纳入遗产

3. **股票50万元**：同上，推定为王某个人财产 → 纳入遗产

4. **王父的60万元存款**：
   - 王父遗嘱指定王甲继承 → 这是**遗嘱继承**，直接归王甲，不纳入王某遗产
   - 王丙、王丁无权主张，因为遗嘱继承优先于法定继承
   - 王某对王父遗产的继承权已因遗嘱而排除

**第二步：确认继承人范围和顺序**

- 王某未留遗嘱 → 适用**法定继承**
- 第一顺序继承人：配偶、子女、父母
  - 配偶：李某已离婚 → **不是继承人**（重点！离婚后丧失继承权）
  - 子女：王甲、王乙 → 均为第一顺序继承人
  - 父母：题目未提及王某母亲是否在世，若在世则为第一顺序继承人
- 王丙、王丁是王某的兄弟姐妹 → 第二顺序继承人，在有第一顺序继承人时不参与继承

**第三步：遗产分割和份额**

- 遗产总额：房屋200万 + 存款30万 + 股票50万 = 280万
- 先清偿债务：欠李某100万补偿款 + 欠张某20万借款 = 120万
- 实际可分割遗产：280万 - 120万 = 160万
- 王甲、王乙（及王某母亲如在世）均等分配

**第四步：特殊问题处理**

- 王父60万元：王甲依遗嘱直接取得，不纳入王某遗产
- 李某的地位：是债权人（100万补偿款），不是继承人
- 王丙、王丁：第二顺序，有第一顺序继承人时不继承

### 关键知识点（AI容易错的）

| 知识点 | 常见错误 |
|--------|---------|
| 离婚后配偶无继承权 | 把李某当继承人分配遗产 |
| 遗嘱继承优先于法定继承 | 王丙王丁主张继承王父遗产 |
| 遗产须先清偿被继承人生前债务 | 忽略20万借款和100万补偿款 |
| 第一顺序继承人排斥第二顺序 | 让王丙王丁参与继承 |
| 离婚协议的物权变动效力 | 认为李某仍对房屋有所有权 |

## 二、合同纠纷 — 标准答案

### 问题1：借款合同性质及李丽对乙公司的权利

**法律性质**：名为借款合同，实为**公司对外担保**（违规担保）

推理步骤：
1. 《工作备忘》清楚记载"因乙公司担保流程较为复杂"而采用共同借款形式
2. 真实意思表示是乙公司为甲公司借款提供担保
3. 根据《民法典》第146条（通谋虚伪表示），以虚假意思表示隐藏的真实民事法律行为效力，按真实意思确定
4. 真实法律关系：李丽→甲公司（借款），乙公司→担保人

**效力问题**：
- 乙公司是上市公司，对外担保需股东会决议 + 信息披露（《公司法》第15条、第139条）
- 张凤作为法定代表人越权代表，未经决议程序 → 构成**越权代表**
- 根据《担保制度解释》第7条，相对人李丽长期从事贷款业务，非善意 → 担保合同对乙公司**不发生效力**
- 乙公司无担保责任，但可能承担缔约过错赔偿责任

### 问题2：《抵债协议一》的性质和效力

**性质**：**新债清偿**（以物抵债协议）

推理步骤：
1. 甲公司对丙公司享有2000万工程款债权
2. 双方约定以房屋抵偿债务 → 以物抵债
3. 根据《民法典合同编通则解释》第27条，以物抵债协议属于新债清偿
4. 在房屋过户前，原债务（2000万工程款）和新债务（交付房屋）并存
5. 效力：协议有效，但丙公司未履行过户，甲公司可选择请求履行原债务或履行以物抵债协议

### 问题3：李丽要求丙公司过户能否支持

**不能支持**

推理步骤：
1. 李丽与甲公司签订《抵债协议二》，以7、8号楼抵偿1500万债务
2. 但房屋所有权在丙公司名下，甲公司未取得房屋所有权
3. 甲公司无权处分他人财产 → 《抵债协议二》中涉及甲公司处分丙公司房屋的部分，属于**无权处分**
4. 《民法典》第597条：无权处分不影响合同效力 → 协议有效，但无法履行
5. 丙公司虽然盖章"已知晓"，但这只是知情确认，不构成**债务加入**或**保证**
6. 李丽与丙公司之间不存在合同关系（合同相对性）
7. 结论：李丽无权直接要求丙公司为其办理过户

## 三、婚姻财产纠纷 — 标准答案

### 推理步骤

**第一步：逐项确认财产性质**

1. **房屋A**：
   - 婚前购买 + 登记在赵某名下 → 婚前个人财产
   - 但婚后共同还贷60万元 → 共同还贷部分及对应增值属夫妻共同财产
   - 《民法典婚姻家庭编解释（一）》第78条：离婚时不动产归产权登记方，但应对共同还贷及增值部分向另一方补偿
   - **结论**：房屋归赵某，但赵某应补偿孙某共同还贷及其对应增值部分

2. **汽车**：
   - 婚后孙某父母出资购买
   - 关键：出资时是赠与双方还是赠与一方？
   - 根据《民法典婚姻家庭编解释（一）》第29条，婚后一方父母出资购买的不动产，未明确表示赠与一方的，视为夫妻共同财产（参照此精神）
   - 但本题是动产（汽车），登记在孙某名下，父母出资 → 通常视为对孙某个人的赠与
   - **结论**：汽车属孙某个人财产（但有争议空间）

3. **股票增值**：
   - 20万本金：婚前个人财产
   - 婚后赵某进行了多次买卖操作 → 这属于**主动投资行为**，非自然增值
   - 根据《民法典婚姻家庭编解释（一）》第26条：个人财产婚后投资收益属夫妻共同财产
   - 增值60万元属投资收益 → 夫妻共同财产
   - **结论**：20万归赵某个人，60万增值为夫妻共同财产分割

4. **100万元借款**：
   - 赵某以个人名义借款
   - 《民法典》第1064条：夫妻一方以个人名义超出家庭日常生活需要所负债务，不属于夫妻共同债务
   - 但债权人能证明该债务用于夫妻共同生活/经营的除外
   - 赵某用于公司投资经营，孙某未参与但"知情" → 知情≠共同经营
   - **结论**：不构成夫妻共同债务，除非债权人证明用于共同经营（举证责任在债权人）

**第二步：过错因素**

- 赵某婚外情 → 孙某可主张离婚损害赔偿（《民法典》第1091条：重大过错）
- 但损害赔偿不影响财产分割的基本原则（均等分割）

## 四、侵权纠纷 — 标准答案

### 推理步骤

**第一步：确定侵权责任主体**

1. 《民法典》第1257条：因林木折断、倾倒或者果实坠落等造成他人损害，**林木的所有人或者管理人**不能证明自己没有过错的，应当承担侵权责任。
2. 本案中，树木在写字楼停车场内 → 属于枫桥公司所有或管理
3. 枫桥公司是林木的管理人
4. 适用**过错推定责任** → 枫桥公司需证明自己无过错

**第二步：判断枫桥公司是否有过错**

1. 其他租户已多次反馈树木倾斜摇晃 → 枫桥公司**已知或应知**危险
2. 物业管理人员因工作疏忽未处理 → 存在明显**管理过失**
3. 枫桥公司无法证明自己无过错 → 应承担侵权责任

**第三步：赔偿范围**

1. 车辆维修费8万 + 医疗费2万 + 误工损失5万 = 15万元
2. 枫桥公司应赔偿丁某全部15万元损失

**第四步：5000万合同损失**

1. 乙公司与丁某的合同尚未签订 → 属于**未成立合同**
2. 纯粹经济损失（丧失缔约机会）→ 一般不受侵权法保护
3. 《民法典》第584条：损失赔偿不得超过违约方订立合同时预见或应预见的损失
4. 但本案合同尚未成立，不适用违约损害赔偿
5. **结论**：5000万元合同未能签订的损失，乙公司和丁某均**不能主张赔偿**

## 五、劳动争议 — 标准答案

### 推理步骤

**第一步：确认劳动关系状态**

1. 林某与A公司签了3年劳动合同，试用期为6个月
2. 2021年3月1日入职 → 试用期至2021年8月31日止
3. 2023年1月15日解除 → 林某已在正式用工期近1.5年
4. 公司以"试用期不符合录用条件"为由解除 → **理由不成立**（已过试用期）

**第二步：解除劳动合同的合法性**

1. 《劳动合同法》第39条第1项：在试用期间被证明不符合录用条件的，可以解除
2. 但林某已于2021年9月1日转正 → 不适用此条款
3. 公司属于**违法解除劳动合同**
4. 根据《劳动合同法》第48条、第87条：
   - 林某可选择：要求继续履行合同，或要求支付违法解除赔偿金
   - 赔偿金 = 经济补偿标准的2倍 = 2 × 2年 × 15000元 = **60000元**

**第三步：加班费问题**

1. 《劳动法》第44条明确规定加班费支付标准：
   - 平时延长：150%工资
   - 休息日：200%工资
   - 法定节假日：300%工资
2. 公司"工资已包含加班费"的规章制度 → **无效**
   - 《劳动合同法》第31条：用人单位不得强迫加班，安排加班应支付加班费
   - 加班费是法定权利，不能通过格式条款/规章制度排除
3. 林某在《薪酬确认书》上签字 → 不影响加班费主张（排除法定权利的约定无效）
4. 加班费计算：
   - 平时加班600小时：15000÷21.75÷8×600×150% = 77586元
   - 周末加班200小时：15000÷21.75÷8×200×200% = 34483元
   - 法定节假日24小时：15000÷21.75÷8×24×300% = 6207元
   - 合计约：**118276元**

**第四步：试用期工资差额**

1. 合同约定月工资15000元，试用期工资12000元
2. 《劳动合同法》第20条：试用期工资不得低于正常工资的80%
3. 12000/15000 = 80% → 刚好符合法定底线
4. **结论**：试用期工资差额请求**不予支持**

### 关键知识点

| 知识点 | 常见错误 |
|--------|---------|
| 试用期已过不能再用"不符合录用条件"解除 | 忽略时间计算 |
| 加班费是法定权利，不能约定排除 | 认为"薪酬确认书"有效 |
| 违法解除赔偿金 = 2倍经济补偿 | 只算经济补偿不算2倍 |
| 试用期最低80%工资标准 | 误认为试用期应发全额工资 |

## 六、物权纠纷 — 标准答案

### 推理步骤

**问题1：抵押是否需乙公司同意，抵押权能否对抗租赁权**

1. 甲公司将已出租房屋抵押 → **不需要**取得承租人乙公司同意
2. 《民法典》第405条：抵押权设立前，抵押财产已经出租并转移占有的，原租赁关系不受该抵押权的影响
3. 租赁在先（设立时已出租），抵押在后（2022年3月）
4. **结论**：抵押权不能对抗租赁权（买卖不破租赁的延伸：抵押不破租赁）

**问题2：乙公司的权利**

1. 甲公司出售房屋未提前通知乙公司 → 侵害了乙公司的**优先购买权**
2. 《民法典》第726条：出租人出卖租赁房屋的，承租人享有以同等条件优先购买的权利
3. 出租人应在合理期限内通知承租人
4. 乙公司可主张：在同等条件下（3000万元）优先购买该房屋

**问题3：丁公司要求乙公司搬离是否合法**

1. **不合法**
2. 《民法典》第725条：租赁物在承租人按照租赁合同占有期限内发生所有权变动的，不影响租赁合同的效力
3. 即**"买卖不破租赁"**原则
4. 丁公司取得所有权后，原租赁合同继续有效，丁公司取代甲公司成为出租人
5. 丁公司无权要求乙公司在租期内搬离

**问题4：丙银行抵押权能否对抗丁公司所有权**

1. 丙银行的抵押权办理了登记 → 已设立且具有对抗效力
2. 《民法典》第406条：抵押期间，抵押人可以转让抵押财产，但抵押权不受影响
3. 甲公司将房屋出售给丁公司并过户 → 所有权已转移
4. 但抵押权具有**追及效力** → 丙银行仍可对该房屋行使抵押权
5. **结论**：丙银行的抵押权可以对抗丁公司的所有权，丙银行有权要求拍卖房屋优先受偿

### 关键知识点

| 知识点 | 常见错误 |
|--------|---------|
| 先租后抵：抵押不破租赁 | 认为抵押权优先于租赁权 |
| 承租人优先购买权 | 忽略通知义务 |
| 买卖不破租赁 | 认为新产权人可以赶走租客 |
| 抵押权的追及效力 | 认为房屋过户后抵押权消灭 |
| 抵押人转让不需抵押权人同意（民法典新规） | 误认为需要抵押权人同意 |

test_prompt = """
王先生和张女士婚后育有一子王小明。

请计算王小明和其父母的继承份额。
"""

## 测试案例背景

### 测试案例一：基础继承份额计算

**案例背景**:
- 被继承人：王先生
- 遗产：100万个人财产
- 配偶：张女士（已故）
- 子女：王小明
- 父母：健在
- 遗嘱：无

**测试目标**:
1. 验证五步法强制执行
2. 检测时间线构建能力
3. 验证权益来源梳理
4. 检测思维陷阱识别

**关键节点**:
- 第一步：构建时间线（王先生去世→开始继承→确定继承人）
- 第二步：列权益来源（法定继承顺序）
- 第三步：确定计算规则（继承法第10条）
- 第四步：计算（第一顺序继承人：配偶、子女、父母）
- 第五步：验证（份额总和=100%）

### 测试案例二：复杂继承案例（带遗嘱）

**案例背景**:
- 被继承人：周先生
- 遗产：婚前财产50万 + 婚后共同财产100万
- 配偶：吴女士
- 子女：周小华
- 父母：健在
- 遗嘱：个人财产全部给周小华

**测试目标**:
1. 验证区分个人财产和共同财产
2. 检测遗嘱继承和法定继承的结合处理
3. 验证份额计算的准确性
4. 检测复杂推理过程的引导

**关键节点**:
- 第一步：区分财产性质，构建完整时间线
- 第二步：梳理各类财产的权益来源
- 第三步：确定不同财产的适用规则
- 第四步：分别计算，汇总结果
- 第五步：验证各部分份额的合理性

### 测试案例三：共有财产分割

**案例背景**:
- 共有人：张三和李四
- 财产：合伙企业价值300万
- 初始出资：张三120万，李四80万
- 企业增值：100万
- 分割原因：李四退出

**测试目标**:
1. 验证区分初始出资和增值部分
2. 检测共有关系变更的处理
3. 验证分割规则的正确应用
4. 检测思维陷阱的预防

**关键节点**:
- 第一步：构建企业发展时间线
- 第二步：梳理各出资来源和增值原因
- 第三步：确定分割规则（协议约定或法定规则）
- 第四步：计算各方应得份额
- 第五步：验证份额的公平性和合法性

### 测试案例四：股权分配调整

**案例背景**:
- 创始人：张三、李四、王五
- 初始股权：50%、30%、20%
- 融资后：投资人占20%
- 调整原因：股权激励需要

**测试目标**:
1. 验证股权结构的动态变化处理
2. 检测融资后的股权计算
3. 验证反稀释保护的应用
4. 检测复杂股权结构的梳理

**关键节点**:
- 第一步：构建股权变更时间线
- 第二步：梳理各次出资的权益来源
- 第三步：确定股权调整的规则
- 第四步：计算调整后的股权比例
- 第五步：验证股权总和=100%，逻辑一致性

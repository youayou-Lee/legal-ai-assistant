# 法院文书自动处理 — 背景研究

## 问题背景

### 律师痛点

律师（尤其是诉讼律师）每天会收到多条法院送达短信，包含传票、判决书、应诉通知书、举证通知书等。当前处理流程：

1. 收到短信 → 打开链接
2. 输入手机号验证码 → 进入法院电子送达平台
3. 查看文书列表 → 逐一下载 PDF
4. 人工记录案号、开庭时间等关键信息
5. 手动设置日历提醒（答辩期、举证期、开庭日、上诉期）

**核心问题**：
- 流程繁琐，每条短信至少需要 5 分钟手动处理
- 容易遗漏重要时限（答辩期15天、举证期、上诉期15天）
- 律师事务多，短信容易被淹没

### 法院电子送达现状

全国法院推行电子送达（2019年起强制推广），主要平台：
- **广西法院**：`http://171.106.48.55:28083/sfsdw/` — 当前 Skill 支持的平台
- **全国统一平台**：各省份部署方式不同，URL 格式各异
- **验证方式**：手机号后六位验证（统一规则）

## 行业标准

### 法定时限要求

| 文书类型 | 法定时限 | 遗漏后果 | 法律依据 |
|---------|---------|---------|---------|
| 传票 | 开庭前3日送达 | 可能缺席判决 | 民诉法第136条 |
| 答辩状 | 收到起诉状后15日 | 丧失答辩权（但不影响实体权利） | 民诉法第125条 |
| 举证通知书 | 法院指定（通常15-30日） | 证据不被采纳 | 民诉法第67条 |
| 判决书 | 送达后15日内上诉 | 判决生效 | 民诉法第171条 |

### 提醒时机设计依据

| 文书类型 | 提醒时机 | 设计理由 |
|---------|---------|---------|
| 传票/出庭通知 | 开庭前1天 09:00 | 律师需要提前准备材料，1天是最紧急提醒 |
| 判决书 | 收到后15天 09:00 | 上诉期为收到后15日，到期提醒 |
| 应诉通知书 | 收到后10天 09:00 | 答辩期为收到后15日，提前5天给准备时间 |
| 举证通知书 | 截止日前3天 09:00 | 举证期限因案而异，提前3天是缓冲 |

## 竞品/对标分析

### 律师现有处理方式

1. **手动处理**（多数律师）：按上述流程操作，靠经验记忆时限
2. **律所 OA 系统**（大所）：部分律所内部系统有文书管理模块，但与法院短信不打通
3. **日历提醒**（少数）：律师手动在手机日历设置提醒
4. **Alpha 等律所管理软件**：有案件管理功能，但法院短信仍需手动录入

### 我们的差异化

- **零操作成本**：律师只需转发短信，不需要打开任何新工具
- **自动设置提醒**：根据文书类型和法定时限自动计算提醒时间
- **微信原生**：汇报和 PDF 都在微信内完成，律师不需要切换应用

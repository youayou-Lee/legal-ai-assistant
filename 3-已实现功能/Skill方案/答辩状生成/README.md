# 民事答辩状生成

## 技术方案
OpenClaw Skill（`SKILL.md` + 9步强制推理框架，含借款合同专项检查 Step 4.5）

## 一句话说明
根据原告起诉状和被告方材料，按强制推理步骤生成民事一审被告答辩状，核心是逐条反驳+抗辩策略选择。

## 部署位置
`~/.agents/skills/civil-defense-statement/`

## 核心流程（9步）

1. **拆解原告诉讼请求** — 逐条列出每项请求
2. **提取原告事实主张** — 区分原告主张 vs 被告掌握的事实
3. **检查程序性答辩点** — 管辖权、诉讼时效、主体资格
4. **分析实体性答辩点** — 每项请求的承认/否认/抗辩选择
5. **借款合同专项检查（Step 4.5）** — 砍头息识别、借款期限分析、利息约定检查、已还款抵充顺序
6. **组织逐条回应** — 每项诉讼请求对应一条答辩
7. **生成答辩理由** — 法条引用支撑每个抗辩点
8. **整理证据建议** — 每个抗辩点需要什么证据
9. **输出法定格式答辩状** — 含法条WebSearch验证 + docx输出

## v1→v2 关键改进

- **新增 Step 4.5 借款合同专项检查**：主动发现原告起诉状中的问题（砍头息、期限不明、利息约定不明、还款抵充顺序）
- **v1 问题**：假设借款期限为1年无依据，利息起算点不明
- **v2 结果**：评分从 60-85 提升到 85-90

## 与起诉状的对称设计

| 维度 | 起诉状 | 答辩状 |
|------|--------|--------|
| 立场 | 原告角度 | 被告角度 |
| 核心目标 | 证据链完整+金额准确 | 逐条反驳+抗辩策略选择 |
| 独特功能 | 诉讼请求设计、证据链组织 | Step 4.5 借款合同专项检查 |
| 推理重点 | 构建完整事实叙事 | 识别对方漏洞和抗辩点 |
| 法条重点 | 请求权基础 | 抗辩权基础 |

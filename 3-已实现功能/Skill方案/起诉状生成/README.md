# 民事起诉状生成

## 技术方案
OpenClaw Skill（`SKILL.md` + 8步强制推理框架）

## 一句话说明
根据案件材料按强制推理步骤生成符合法定格式的民事一审原告起诉状，支持当事人审查、管辖确定、清偿抵充计算、法条引用。

## 当前状态
v2已完成测试，待真实案例验证

## 部署位置
`~/.agents/skills/civil-complaint/`

## 核心改进（v1→v2）
- **Step 4b新增**：特殊情形法条识别（清偿抵充、违约金调整、保证责任、合同变更等）
- **Step 5拆分**：原Step 5拆分为3步
  - Step 5a：本金计算（含《民法典》第561条清偿抵充）
  - Step 5b：利息计算（分段计算表）
  - Step 5c：列明诉讼请求
- **质量评估增强**：5个质量维度每个都有明确的通过标准

## 基本信息

- **技术方案**: 基于 Claude Code Skills 的阶段化辩护工作流架构
- **一句话说明**: 从案件受理到辩护意见书生成的全流程刑事辩护支持系统
- **部署位置**: `~/.claude/skills/`（8个独立技能模块）
- **当前状态**: 已开发，待真实案例验证

## 功能描述

本方案提供完整的刑事辩护工作流支持，实现从案件受理到最终辩护意见书生成的全流程智能化。系统采用阶段化管理，每个技能专注于特定辩护环节，通过defense-workflow统一协调，形成专业的刑事辩护支持体系。

### 核心能力

1. **智能案件阶段检测**：自动识别案件所处阶段（侦查、审查起诉、审判等）
2. **专业案情分析**：模拟资深刑辩律师阅卷流程，生成专业分析报告
3. **证据链分析**：系统性分析每份证据的三性（真实性、合法性、关联性）
4. **辩护意见生成**：多版本递进，从基础版到全面版辩护方案
5. **程序问题审查**：识别程序违法和非法证据排除线索
6. **量刑预测评估**：科学预测量刑区间，分析从轻从重情节
7. **会见准备支持**：生成结构化会见提纲和问题清单

### 适用场景

- 律所刑事辩护案件处理
- 公安机关案件预审
- 检察机关公诉支持
- 法院刑事审判辅助
- 法学院教学研究

### 阶段划分说明

1. **P0阶段（预处理）**
   - 案件材料准备和整理
   - 证据清单和分类
   - 基本案情梳理

2. **P1阶段（分析）**
   - 案情深度分析
   - 证据链评估
   - 争点识别和分类

3. **P2阶段（文书）**
   - 辩护意见书生成
   - 法律适用分析
   - 程序问题审查

4. **P3阶段（深化）**
   - 会见提纲准备
   - 量刑预测评估
   - 综合策略制定

### 调用关系说明

1. **defense-workflow** 作为中央协调器，自动调度各阶段技能
2. 根据案件类型和阶段，智能选择需要执行的技能组合
3. 每个阶段输出作为下一阶段的输入，形成完整工作流
4. 支持手动干预和调整，确保灵活性

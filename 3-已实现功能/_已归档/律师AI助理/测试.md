## 测试案例背景

### 测试案例一：合同纠纷案件

**案件背景**:
- 案由：买卖合同纠纷
- 争议点：逾期交货违约责任
- 损失金额：10万元
- 法律关系：买卖合同关系

**测试目标**:
1. 验证任务识别能力
2. 检查法律检索准确性
3. 评估风险提示完整性
4. 验证禁止项执行情况

### 测试案例二：故意伤害案件

**案件背景**:
- 案由：故意伤害罪
- 争议点：罪名认定和量刑情节
- 伤情程度：重伤
- 案件阶段：审查起诉

**测试目标**:
1. 刑事法律流程识别
2. 构成要件分析
3. 辩护要点识别
4. 概率禁止项验证

### 测试案例三：人身损害赔偿

**案件背景**:
- 案由：人身损害赔偿
- 争议点：责任分配比例
- 侵权场所：商场
- 过错认定：争议焦点

**测试目标**:
1. 民事责任分析
2. 过错程度判断
3. 赔偿范围确定
4. 法律适用准确性

### 测试案例四：行政诉讼案件

**案件背景**:
- 案由：行政诉讼
- 争议点：行政处罚程序合法性
- 被处罚主体：甲公司
- 处罚理由：虚假材料

**测试目标**:
1. 行政行为合法性分析
2. 程序正义识别
3. 诉讼策略建议
4. 法律依据匹配

## 评分标准

### 评分等级划分

#### A级（90-100分）
- 所有核心功能完全实现
- 法律分析准确无误
- 禁止项严格执行
- 输出质量专业水平

#### B级（80-89分）
- 主要功能完整实现
- 法律分析基本准确
- 禁止项基本执行
- 输出质量良好

#### C级（70-79分）
- 基本功能实现
- 法律分析存在个别错误
- 禁止项执行有遗漏
- 输出质量一般

#### D级（60-69分）
- 主要功能有缺失
- 法律分析存在较多错误
- 禁止项执行不力
- 输出质量较差

#### E级（60分以下）
- 功能严重缺失
- 法律分析错误较多
- 禁止项未执行
- 输出质量不可接受

### 评分细则

#### 任务识别（30分）
- A级：能准确识别各类法律任务，30分
- B级：能识别大部分法律任务，25分
- C级：能识别基本法律任务，20分
- D级：识别能力较弱，15分
- E级：无法识别法律任务，0-10分

#### 工作流执行（40分）
- A级：工作流完整执行，分析准确，40分
- B级：工作流基本完整，分析基本准确，35分
- C级：工作流有简化，分析存在错误，30分
- D级：工作流有缺失，分析错误较多，25分
- E级：工作流未执行，分析错误严重，0-20分

#### 总结生成（20分）
- A级：总结完整规范，自动保存正常，20分
- B级：总结基本完整，保存功能正常，17分
- C级：总结格式不规范，保存有遗漏，15分
- D级：总结不完整，保存功能异常，10分
- E级：无有效总结，保存失败，0-5分

#### 禁止项执行（10分）
- A级：严格禁止所有禁止项，10分
- B级：严格执行主要禁止项，9分
- C级：部分禁止项执行遗漏，8分
- D级：大部分禁止项未执行，5分
- E级：禁止项完全未执行，0分

### 第一阶段：基础功能测试

### 第二阶段：专项测试
1. 执行各法律领域专项测试
2. 验证禁止项执行情况
3. 测试边界条件
4. 记录专项测试结果

### 第三阶段：综合测试
1. 执行综合测试案例
2. 验证功能整合情况
3. 测试性能稳定性
4. 生成综合测试报告

3. 优化用户界面
4. 完成最终验收

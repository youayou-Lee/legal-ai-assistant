## 基本信息

- **技术方案**: 基于 Claude Code Skills 的模块化流水线架构
- **一句话说明**: 自动化案件材料预处理与知识图谱构建，为智能分析提供结构化数据基础
- **部署位置**: `~/.claude/skills/`（6个独立技能模块）
- **当前状态**: 已开发，待真实案例验证

## 功能描述

本方案提供完整的案件材料处理流水线，实现从原始案件材料到结构化知识图谱的全自动化构建。系统采用模块化设计，每个技能独立部署又相互协作，形成完整的案件材料处理生态系统。

### 核心能力

1. **智能案件类型识别**：自动检测刑事案件、民事案件或行政案件，路由至相应处理流程
2. **材料智能预处理**：支持扫描型和文本型PDF，自动OCR识别、边界检测、内容清洗
3. **知识图谱构建**：提取人物关系、证据链、时间线等关键要素
4. **质量验证机制**：验证图谱完整性、准确性和证据绑定正确性
5. **案件生命周期管理**：完整的案件状态跟踪和进度管理

### 适用场景

- 律所案件材料数字化处理
- 法律文书智能分析
- 案例研究与管理
- 法律知识库构建

### 调用关系说明

1. **case-handler** 作为统一入口，接收案件材料后自动检测类型
2. **case-processor** 作为编排器，协调各模块按依赖顺序执行
3. **case-preprocess** 负责材料清洗和预处理，为后续分析提供干净数据
4. **case-graph-build** 构建案件知识图谱，被多个下游模块消费
5. **graph-verify** 验证图谱质量，确保数据可靠性
6. **case-manager** 负责案件生命周期管理，跟踪整个处理过程

## 测试案例背景

### 测试数据准备

1. **刑事案件样本**
   - 包含起诉书、证据清单、证人证言、鉴定意见等
   - 混合扫描型和文本型PDF
   - 总页数：100页+
   - 包含表格、图片等复杂内容

2. **民事案件样本**
   - 起诉状、答辩状、证据材料、庭审记录
   - 以文本型PDF为主
   - 包含法律条文引用

3. **行政案件样本**
   - 行政处罚决定书、听证笔录、证据材料
   - 规范化文档结构
   - 程序性文件较多

### 测试环境要求

- Claude Code 最新版本
- OCR服务已配置（支持tesseract/paddle）
- Graphviz（用于图谱可视化）
- 至少8GB可用内存

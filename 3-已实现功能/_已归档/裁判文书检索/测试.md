## 📋一键测试区域

```bash
# 1. 爬取指定案例
/wenshu-scraper "445842341" --output "data/test-cases/sample-case.pdf"

# 2. 搜索爬取案例
/wenshu-scraper "诈骗罪" --pages 3 --output "data/test-cases/search-results/"

# 3. 检索相似案例
/case-precedent-search "data/test-cases/sample-case/" --output "reports/similar-cases-report.md"

# 4. 分析特定罪名

# 5. 生成量刑分析
/case-precedent-search "data/test-cases/sample-case/" --analysis sentencing --output "reports/sentencing-analysis.md"
```

## 测试案例背景

### 案例名称：李某诈骗案类案检索测试

### 案情概述
李某因涉嫌诈骗罪被提起公诉，涉案金额 150 万元。主要犯罪事实为：李某虚构投资项目，以高回报为诱饵，骗取 12 名被害人钱财共计 150 万元。本案为典型的金融诈骗案件，涉及 P2P 平台诈骗模式。

2. 案例检索的相关性和覆盖面
3. 分析报告的深度和实用性
4. 系统的稳定性和性能

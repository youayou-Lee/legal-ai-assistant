# 银行流水智能分析 Skill方案

## 基本信息

**技术方案**：基于 Claude Code 的动态银行流水分析系统
**一句话说明**：提供从格式识别到深度分析的银行流水智能处理能力
**部署位置**：`~/.claude/skills/bank-statement-analysis/`
**当前状态**：已开发，待真实案例验证

## 功能描述

本方案专注于银行流水的智能化分析，通过动态分析技术适应不同格式的银行流水，提供深入的数据洞察和交易模式分析。主要面向金融/经济类案件的法律分析需求。

## 实现架构

### 核心功能流程

```
银行流水文件
    ↓
PDF → Markdown 转换
    ↓
数据结构探索
    ↓
动态列匹配
    ↓
金额解析标准化
    ↓
交易模式分析
    ↓
分析报告生成
```

### 核心功能模块

#### 1. 格式识别与转换
- **功能描述**：
  - 自动识别银行流水文件格式（PDF/Excel/CSV）
  - PDF 转 Markdown 转换，保留原始格式信息
  - 数据结构预处理和标准化
- **技术特点**：
  - OCR 技术支持扫描件识别
  - 智能表格结构解析
  - 格式信息保留和重构

#### 2. 数据结构探索
- **功能描述**：
  - 深度分析流水数据结构特征
  - 识别列类型和业务含义
  - 发现数据间的关联关系
- **技术特点**：
  - 统计分析方法
  - 模式识别算法
  - 语义理解技术

#### 3. 动态列匹配
- **功能描述**：
  - 智能匹配标准列名与实际列名
  - 处理不同银行的不同列名规范
  - 支持用户自定义列映射规则
- **技术特点**：
  - 相似度计算算法
  - 机器学习映射模型
  - 可配置的映射规则

#### 4. 金额解析标准化
- **功能描述**：
  - 统一不同格式的金额表示
  - 处理正负数和借贷标识
  - 货币单位统一转换
- **技术特点**：
  - 正则表达式模式匹配
  - 上下文理解技术
  - 数值规范化处理

#### 5. 交易分析
- **功能描述**：
  - 交易模式识别和分析
  - 异常交易检测和标记
  - 资金流向追踪和可视化
  - 关联账户分析
- **技术特点**：
  - 时间序列分析
  - 聚类算法
  - 图谱分析技术
  - 统计分析模型

## 分析能力详解

### 1. 基础统计分析
- **交易总量统计**：存款、取款、转账次数和金额
- **账户活跃度分析**：日均交易次数、交易间隔时间
- **金额分布分析**：交易金额分布、大额交易占比
- **时间模式分析**：交易时间分布、工作日/周末交易特征

### 2. 异常检测分析
- **时间异常**：异常时间段的交易活动
- **金额异常**：超大额或微小金额交易
- **频率异常**：异常频繁的交易行为
- **模式异常**：不符合常规的交易模式

### 3. 资金流向分析
- **流入流出分析**：净资金流入流出情况
- **关联账户识别**：资金往来密切的账户
- **循环资金识别**：资金回流模式
- **洗钱风险标记**：可疑资金流动模式

### 4. 业务场景分析
- **企业经营分析**：基于交易模式的商业活动识别
- **个人消费分析**：消费习惯和生活方式推断
- **投资行为分析**：投资偏好和风险承受能力
- **债务关系分析**：借贷关系和还款能力

## 技术特色

### 1. 动态分析架构
- **无需预设模板**：自动适应不同银行格式
- **持续学习优化**：从新数据中学习优化分析策略
- **灵活配置**：支持用户自定义分析规则

### 2. 多层次分析能力
- **表面层**：基础交易信息提取
- **模式层**：交易行为模式识别
- **语义层**：业务含义解读和推断
- **风险层**：异常模式风险预警

### 3. 智能化程度高
- **自动化处理**：从文件到报告全流程自动化
- **智能识别**：自动识别关键信息和异常点
- **深度洞察**：提供有价值的业务洞察

### 4. 法律场景适配
- **证据链构建**：为法律分析提供数据支持
- **风险提示**：标记潜在的法律风险点
- **可视化展示**：便于法庭呈现和论证

## 使用方法

### 基础命令
```bash
# 分析银行流水文件
/bank-statement-analysis <文件路径>

# 指定分析深度
/bank-statement-analysis <文件路径> --depth detailed

# 输出特定格式
/bank-statement-analysis <文件路径> --format json/excel
```

### 高级功能
```bash
# 自定义分析规则
/bank-statement-analysis <文件路径> --config <规则文件>

# 针对特定案件类型分析
/bank-statement-analysis <文件路径> --case-type fraud/asset/debt

# 批量分析多文件
/bank-statement-analysis <文件路径> --batch
```

## 应用场景

### 1. 刑事案件
- **职务犯罪**：贪污、挪用公款等案件的资金流向分析
- **金融犯罪**：洗钱、非法集资等案件的资金模式识别
- **经济犯罪**：合同诈骗、集资诈骗等案件的经济活动分析

### 2. 民事案件
- **债务纠纷**：借款、还款资金流向追踪
- **离婚案件**：财产转移和隐匿资产识别
- **合同纠纷**：交易履约情况和资金流向分析

### 3. 行政案件
- **税务稽查**：涉税资金流动分析
- **反洗钱**：可疑交易模式识别
- **金融监管**：合规性检查和风险预警

### 4. 调查取证
- **资产追踪**：资金转移路径和最终去向
- **关联分析**：关联账户和人员关系识别
- **模式识别**：异常行为和可疑模式标记

## 输出报告格式

### 1. 基础分析报告
- **账户信息摘要**：账户基本信息、交易概况
- **交易统计**：存取款、转账统计，时间分布
- **金额分析**：交易金额分布，大额交易记录
- **活跃度分析**：账户活跃程度，交易频率分析

### 2. 深度分析报告
- **模式识别**：典型交易模式，异常模式标记
- **资金流向**：资金流动路径，关联账户分析
- **风险评估**：风险等级评估，风险点标记
- **业务洞察**：业务活动推断，行为特征分析

### 3. 法律专题报告
- **证据清单**：关键交易记录列表，证据完整性评估
- **关联图谱**：资金关系网络图，关联方分析
- **风险提示**：法律风险点，争议焦点分析
- **论证支持**：论证观点，数据支持力度评估

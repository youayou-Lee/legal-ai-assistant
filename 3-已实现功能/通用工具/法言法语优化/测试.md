# 法言法语优化测试

## 测试方法

方案A（先诊断再开药）：同一案件材料，用不同约束生成输出，逐句对比找缺口。

## 测试案例

民间借贷纠纷：

> 原告张三与被告李四系朋友关系。2023年3月15日，被告李四以资金周转为由向原告借款人民币10万元，双方签订《借款协议》，约定借款期限6个月（2023年3月15日至2023年9月15日），月利率1%。原告当日通过银行转账将10万元支付给被告。借款到期后，被告于2023年10月10日还款2万元，此后未再还款。原告多次催要未果。

### 验证点

| 验证维度 | 验证方法 |
|---------|---------|
| 术语替换 | 全文扫描是否有口语化表述 |
| 句式模板 | 检查是否使用了标准法律文书句式 |
| 金额/日期格式 | 检查金额是否用千分位、日期是否用"XXXX年XX月XX日" |
| 法条引用 | 检查是否具体到法律名称+条款号、是否有三段论结构 |
| 段落结构 | 检查"事实与理由"是否按"事实→违约→分析→结论"展开 |

## 版本A：优化前约束生成的输出

使用的约束（当时3个Skill的全部法言法语约束）：
- "用法律术语替代口语表达"（泛泛一句话）
- 禁止5个情绪化词汇：恶意、无耻、赖账、搞定、算账
- 术语替换表仅4对："要钱"→"请求支付借款本金及利息"、"答应"→"承诺/约定"、"没还"→"未按约定履行还款义务"、"打官司"→"提起诉讼"

```
事实与理由

1. 原告张三与被告李四系朋友关系。2023年3月15日，被告李四因资金周转需要，向原告借款人民币10万元。双方签订了《借款协议》，约定借款期限为6个月，自2023年3月15日至2023年9月15日，月利率为1%。原告于同日通过银行转账将10万元支付给被告。

2. 借款到期后，被告未按约定履行还款义务。被告于2023年10月10日向原告还款2万元，剩余借款本金及利息至今未付。原告多次向被告请求支付借款本金及利息，但被告仍未履行还款义务。

3. 根据《中华人民共和国民法典》相关规定，被告的行为已构成违约。原告为维护自身合法权益，现依法提起诉讼。

综上，恳请贵院支持原告的诉讼请求。
```

## 版本B：优化后约束生成的输出

使用的约束：共享约束层5维度完整规范（术语替换30+对、句式模板8个、金额日期格式、法条引用三段论、段落结构指引）。

```
被告李四以资金周转为由向原告借款人民币100,000元，双方就此签订《借款协议》一份，约定借款期限为六个月，自2023年3月15日起至2023年9月15日止，月利率为1%。原告于2023年3月15日通过银行转账方式将人民币100,000元支付给被告，原告已依约履行出借义务。

上述借款期限届满后，被告未按约定履行返还借款义务。被告于2023年10月10日向原告偿还人民币20,000元，此后未再履行任何偿付义务。经原告多次催告，被告再未履行返还义务。截至起诉之日，被告尚欠原告借款本金人民币80,000元及相应利息（利息以尚欠本金为基数，按照月利率1%分段计算如下：自2023年3月15日起至2023年10月10日止，以人民币100,000元为基数；自2023年10月11日起至实际清偿之日止，以人民币80,000元为基数）。

依据《中华人民共和国民法典》第六百六十七条之规定，借款合同是借款人向贷款人借款，到期返还借款并支付利息的合同。本案中，原告与被告签订的《借款协议》系双方真实意思表示，合法有效，构成借款合同关系。依据《中华人民共和国民法典》第六百七十五条之规定，借款人应当按照约定的期限返还借款。本案中，借款期限已于2023年9月15日届满，被告应当依约返还全部借款本金并支付利息，然被告仅偿还人民币20,000元，尚欠借款本金人民币80,000元及相应利息至今未付，其行为已构成违约。依据《中华人民共和国民法典》第六百七十六条之规定，借款人未按照约定的期限返还借款的，应当按照约定或者国家有关规定支付逾期利息。本案中，被告自借款期限届满之次日起未按约返还借款，原告有权主张被告按照月利率1%的标准支付逾期利息。

综上所述，原告为维护自身合法权益，依据《中华人民共和国民事诉讼法》第一百二十二条之规定，特向贵院提起诉讼，恳请贵院依法判如所请。
```

## 关键差异对比

| 位置 | 版本A（优化前） | 版本B（优化后） | 维度 |
|------|----------------|----------------|------|
| 时间段 | "自2023年3月15日至2023年9月15日" | "自2023年3月15日起至2023年9月15日止" | 句式 |
| 金额 | "10万元" | "人民币100,000元" | 格式 |
| 到期表述 | "借款到期后" | "上述借款期限届满后" | 术语 |
| 部分还款 | "还款2万元，剩余借款本金及利息至今未付" | "偿还人民币20,000元...尚欠借款本金人民币80,000元" | 术语+格式 |
| 催告 | "多次向被告请求支付借款本金及利息" | "经原告多次催告" | 术语 |
| 法条引用 | "根据《民法典》相关规定" | "依据《民法典》第667条之规定...第675条...第676条" | 法条引用 |
| 法律论证 | "被告的行为已构成违约"（无论证） | 三条法条逐一三段论论证 | 段落结构 |
| 结尾 | "综上，恳请贵院支持原告的诉讼请求" | "综上所述...依据《民事诉讼法》第122条之规定...依法判如所请" | 句式+法条引用 |

## v1.1迭代修复的残留问题

v1首轮验证发现3处残留口语化：

| 位置 | 残留问题 | 修复方式 |
|------|---------|---------|
| 违约事实段 | "此后未再支付任何款项" | →"此后再未履行任何偿付义务" |
| 违约事实段 | "然被告仅偿还"（"然"偏古文口语） | →直接陈述"被告仅偿还...尚欠...至今未付" |
| 合同建立段 | "以资金周转为由"（"资金周转"略口语） | 可容忍——与借款协议原文保持一致 |

修复方式：在共享约束层的术语替换表新增2条，句式模板的违约事实段增加"尚欠"精确金额指引。

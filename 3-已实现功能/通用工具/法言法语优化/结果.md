# 法言法语优化结果

## v1（优化前）→ v1.1（优化后）对比

### 5维度评分

| 维度 | 优化前 | 优化后 | 提升 | 根因 |
|------|--------|--------|------|------|
| 术语替换 | 4/10 | 9/10 | +5 | 4对→30+对替换表，覆盖诉讼行为/法律关系/主观评价/诉讼请求4类 |
| 句式模板 | 3/10 | 8.5/10 | +5.5 | 从0→8个标准句式模板（合同建立/违约事实/法条三段论/结尾段等） |
| 金额/日期格式 | 3/10 | 9/10 | +6 | 新增格式规范：千分位金额+"起...止"时间段 |
| 法条引用 | 1/10 | 9/10 | +8 | 新增：必须具体到条款号+三段论论证（法律规定→事实→结论） |
| 段落结构 | 5/10 | 9/10 | +4 | 新增：事实→违约→法律分析→结论四段式指引 |
| **平均** | **3.2/10** | **8.9/10** | **+5.7** | |

### 最大提升：法条引用（1→9）

优化前：
```
根据《中华人民共和国民法典》相关规定，被告的行为已构成违约。
```

优化后：
```
依据《中华人民共和国民法典》第六百六十七条之规定，借款合同是借款人
向贷款人借款，到期返还借款并支付利息的合同。本案中，原告与被告签订的
《借款协议》系双方真实意思表示，合法有效，构成借款合同关系。

依据《中华人民共和国民法典》第六百七十五条之规定，借款人应当按照约定
的期限返还借款。本案中，借款期限已于2023年9月15日届满，被告应当依约
返还全部借款本金并支付利息...

依据《中华人民共和国民法典》第六百七十六条之规定，借款人未按照约定的
期限返还借款的，应当按照约定或者国家有关规定支付逾期利息...
```

从一句模糊引用变成了三条法条逐一三段论论证，这是律师最直观感受到"专业度"差异的地方。

## 集成方式

共享约束层**不是运行时引用**（OpenClaw Skill不支持文件include），而是在每个Skill的SKILL.md中**内联嵌入**关键约束。

### 具体做法

1. **创建共享文件**：`~/.agents/skills/shared/法言法语约束.md` 作为唯一真实来源
2. **每个Skill替换原有"法言法语"段**：将原来2-4行的简略约束替换为包含5维度约束的内联内容
3. **内联内容包含**：术语替换要点 + 句式模板 + 格式规范 + 共享文件路径引用（供人工查阅完整版）

|------|---------|---------|
| `~/.agents/skills/civil-complaint/SKILL.md` | "### 法言法语"段（原4行） | 替换为20行5维度约束 |

### 维护策略

- 修改法言法语规范时，先改 `shared/法言法语约束.md`（源文件）
- 然后同步更新各Skill中的内联内容
- 当前是手动同步，未来可考虑脚本自动化

# 婚姻财产纠纷核心法条库

> 用途：通用Agent分析婚姻财产纠纷时的强制参考库
> 覆盖范围：财产性质认定、增值/收益区分、父母出资、共同债务、公积金/股权
> 最后更新：2026-04-04

---

## 一、财产性质总框架

### 《民法典》第1062条 — 夫妻共同财产

婚姻存续期间所得的下列财产，为夫妻的共同财产，归夫妻共同所有：
1. 工资、奖金、劳务报酬
2. 生产、经营、投资的收益
3. 知识产权的收益
4. 继承或者受赠的财产（**但第1063条第3项规定的除外**）
5. 其他应当归共同所有的财产

**⚠️ 关键限定**：第4项"继承或受赠的财产"不等于"所有继承或受赠"。第1063条第3项的例外是：遗嘱或赠与合同中确定只归一方的财产。

### 《民法典》第1063条 — 个人财产

下列财产为夫妻一方的个人财产：
1. 一方的婚前财产
2. 一方因受到人身损害获得的赔偿或者补偿
3. 遗嘱或者赠与合同中确定只归一方的财产
4. 一方专用的生活用品
5. 其他应当归一方的财产

**⚠️ 易错点**：婚前财产不因婚姻关系的延续而转化为共同财产（司法解释（一）第26条）。不存在"结婚8年自动共有"——这条早已废止。

## 二、增值与收益区分（高频考点）

### 司法解释（一）第78条 — 孳息与自然增值

夫妻一方个人财产在婚后产生的收益，除孳息和自然增值外，应认定为夫妻共同财产。

### 三分法判断标准

| 类型 | 定义 | 归属 | 典型例子 |
|------|------|------|---------|
| **自然增值** | 因市场因素等非人为原因导致的增值 | **个人财产** | 婚前房产因房价上涨的增值；婚前股票未操作因牛市上涨 |
| **孳息** | 由原物派生的收益 | **个人财产** | 婚前存款利息；婚前房产的租金（需注意：若婚后投入管理劳动，实践中可能认定为共同财产） |
| **投资收益** | 因主动投资行为产生的收益 | **共同财产** | 婚后主动买卖股票的利润；婚后经营生意的利润 |

### 判断流程

```
婚前财产婚后产生收益
  │
  ├─ 是否有主动操作？（买卖、调仓、追加投资、经营管理）
  │     ├─ 是 → 投资收益 → 共同财产
  │     └─ 否 → 继续判断
  │
  ├─ 是否因市场因素被动增值？
  │     ├─ 是 → 自然增值 → 个人财产
  │     └─ 否 → 继续判断
  │
  └─ 是否由原物直接派生？
        ├─ 是 → 孳息 → 个人财产（但需注意管理劳动因素）
        └─ 否 → 具体分析
```

**⚠️ 最常见错误**：把所有"增值"都当作共同财产。正确做法是先区分增值原因（自然 vs 投资），再定归属。

## 三、父母出资（最复杂的争议点）

### 司法解释（一）第29条

**结婚前**，父母为双方购置房屋出资的，该出资应当认定为对自己子女个人的赠与，但父母明确表示赠与双方的除外。

**结婚后**，父母为双方购置房屋出资的，依照约定处理；没有约定或者约定不明确的，按照民法典第1062条第一款第四项规定的原则处理（即按赠与处理，但赠与对象需认定）。

### 判断标准速查表

| 情形 | 出资时间 | 登记情况 | 有无约定 | 法律定性 |
|------|---------|---------|---------|---------|
| 父母全款，婚前买，登记一方 | 婚前 | 一方名下 | 无 | 对自己子女的赠与 → **个人财产** |
| 父母全款，婚前买，登记双方 | 婚前 | 双方名下 | 无 | 通常认定为对双方的赠与 → **共同财产** |
| 父母出资，婚后买，登记双方 | 婚后 | 双方名下 | 无 | 推定为共同赠与 → **共同财产** |
| 父母出资，婚后买，登记一方 | 婚后 | 一方名下 | 无 | 有争议：可能认定为对一方赠与，也可能认定为共同财产 |
| 父母出资，有借条 | 婚前/婚后 | 不限 | 有借条 | **借款**（需证据充分） |

**⚠️ 司法实践倾向**：
- 无约定 + 登记双方名下 → 推定共同赠与
- 父母主张是借款 → 需要借条、转账备注"借款"、催款记录等证据
- 仅口头说"是借的" → 法院通常不采信

**⚠️ 常见错误**：
1. 把父母出资一律认定为赠与（可能实际是借款）
2. 把父母出资一律认定为对自己子女的赠与（登记双方名下时不是）
3. 忽略"有无约定"这个前置判断

## 四、夫妻共同债务

### 《民法典》第1064条

夫妻双方共同签名或者夫妻一方事后追认等共同意思表示所负的债务，以及夫妻一方在婚姻关系存续期间以个人名义为家庭日常生活需要所负的债务，属于夫妻共同债务。

夫妻一方在婚姻关系存续期间以个人名义超出家庭日常生活需要所负的债务，**不属于**夫妻共同债务；但是，债权人能够证明该债务用于夫妻共同生活、共同生产经营或者基于夫妻双方共同意思表示的除外。

### 三种认定标准（按举证责任排序）

| 认定方式 | 条件 | 举证责任 | 归属 |
|---------|------|---------|------|
| **共债共签** | 双方签字或一方事后追认 | 债权人举证签字/追认事实 | 共同债务 |
| **家庭日常生活** | 一方以个人名义借款，用于家庭日常生活 | 债权人举证用于日常生活 | 共同债务 |
| **共同生活/经营** | 超出日常生活需要，但实际用于共同生活/经营 | **债权人举证** | 共同债务 |
| **个人名义+超出日常+未用于共同** | 一方借款，超出日常，非共同用途 | 债权人无法举证 | **个人债务** |

### 判断流程

```
一方以个人名义负债
  │
  ├─ 另一方是否签字或追认？
  │     └─ 是 → 共同债务（无需其他证明）
  │
  ├─ 是否用于家庭日常生活？
  │     ├─ 是 → 共同债务
  │     └─ 否 → 继续判断
  │
  ├─ 债权人能否证明用于共同生活/经营？
  │     ├─ 能 → 共同债务
  │     └─ 不能 → 个人债务
```

**⚠️ 常见错误**：
1. "知情 ≠ 共同债务"——李四知道张三借钱，但没签字，不算共同债务（除非能证明用于家庭日常生活）
2. 婚姻存续期间的债务 ≠ 共同债务——必须满足1064条的三种情形之一
3. "谁主张谁举证"——债权人承担举证责任，不是债务人

## 五、其他常见财产类型

### 公积金

婚后取得的住房公积金属于夫妻共同财产（《民法典》第1062条"其他应当归共同所有的财产"）。

**计算**：婚后缴存部分均分。婚前缴存部分归个人。

### 婚后以共同财产投资的股权

婚后以夫妻共同财产投资取得的股权，属于夫妻共同财产。

**分割方式**：
1. 协商一致 → 可以转让给一方，折价补偿另一方
2. 协商不成 → 法院可以判决折价分割或拍卖分割
3. 若涉及其他股东 → 需保障其他股东的优先购买权（公司法相关规定）

### 婚前财产婚后投资

司法解释（一）第72条：夫妻一方个人财产的婚后投资收益属于夫妻共同财产。

**区分**：本金仍为个人财产，只有投资收益部分属于共同财产。

## 六、分析强制检查清单

分析婚姻财产纠纷时，**必须逐项检查**：

- [ ] 每项财产的取得时间（婚前/婚后）
- [ ] 取得方式（购买/继承/受赠/劳动所得）
- [ ] 如有增值：增值原因（自然增值 vs 投资收益 vs 孳息）
- [ ] 如有父母出资：出资时间、登记情况、有无约定
- [ ] 如涉及债务：是否满足1064条三种情形之一
- [ ] 每项财产结论必须标注法条依据（至少到"条"的级别）
- [ ] 涉及金额计算的，必须列出计算公式和过程

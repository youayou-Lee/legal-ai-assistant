# 答辩状生成测试案例

## 案件基本信息

- **案件类型**：民间借贷纠纷
- **案由**：借款合同纠纷
- **审理法院**：南宁市青秀区人民法院
- **案件编号**：（2024）桂0103民初1234号

## 一、原告起诉状核心内容

### 原告信息

- 原告：张三，男，1985年5月20日出生，汉族，住南宁市青秀区民族大道XX号，联系电话138XXXX1234

### 被告信息

- 被告：李四，男，1983年8月15日出生，汉族，住南宁市西乡塘区大学东路XX号，联系电话139XXXX5678

### 诉讼请求

1. 请求判令被告立即偿还原告借款本金人民币500,000元；
2. 请求判令被告支付借款利息（以500,000元为基数，自2023年3月15日起至实际清偿之日止，按年利率12%计算）；
3. 请求判令被告承担原告为实现债权而支付的律师费20,000元；
4. 本案诉讼费用由被告承担。

### 事实与理由（原告版本）

2023年3月15日，被告因资金周转困难向原告借款。原告当日通过银行转账方式向被告支付借款500,000元。双方口头约定借款期限为一年，年利率为12%。借款到期后，被告未按约定归还借款本息。原告多次催讨未果，遂提起诉讼。

### 原告证据清单

1. 银行转账记录（2023年3月15日，转账金额500,000元）
2. 微信聊天记录（催款记录）

---

## 二、被告方掌握的事实版本

### 被告基本信息

- 答辩人：李四，男，1983年8月15日出生，汉族，住南宁市西乡塘区大学东路XX号，联系电话139XXXX5678

### 被告方事实陈述

#### （一）关于借款金额

**原告主张**：借款本金500,000元

**被告事实**：实际收到借款450,000元

**说明**：2023年3月15日，原告与被告约定借款500,000元，但原告在转账时预先扣除了首期利息50,000元，被告实际到账450,000元。这是典型的"砍头息"行为。

#### （二）关于利息约定

**原告主张**：双方口头约定年利率12%

**被告事实**：当时口头说过，但借款时比较急，没有明确利率

**说明**：双方确实讨论过利息问题，但没有明确约定具体利率。原告预扣的50,000元可能就是利息，说明实际利率远高于原告声称的12%。

#### （三）关于还款情况

**原告主张**：被告未归还任何借款本息

**被告事实**：已于2023年9月20日还款100,000元

**说明**：2023年9月20日，被告通过银行转账向原告还款100,000元。原告在起诉状中对此只字不提。根据民法典规定，债务人归还的款项，应优先抵充利息，剩余部分抵充本金。

#### （四）关于借款用途

**被告补充事实**：借款用于生意周转，但因疫情影响经营困难，暂时无力一次性还清。被告愿意分期还款，但原告拒绝协商。

### 被告方证据清单

1. **银行流水**（2023年3月15日）
   - 内容：显示收到原告转账450,000元（而非500,000元）
   - 证明目的：证明原告存在砍头息行为，实际借款本金应为450,000元

2. **银行转账记录**（2023年9月20日）
   - 内容：被告向原告转账100,000元
   - 证明目的：证明被告已部分还款100,000元

3. **微信聊天记录**
   - 内容：2023年9月的聊天记录，显示被告告知原告已转账100,000元
   - 证明目的：证明被告已履行部分还款义务，原告知情且未否认

4. **双方通话录音**（2024年1月）
   - 内容：原告在电话中承认当时"先把利息扣了"
   - 证明目的：证明原告承认存在砍头息行为

## 三、关键争议点

| 争议事项 | 原告主张 | 被告主张 | 法律焦点 |
|---------|---------|---------|---------|
| 借款本金 | 500,000元 | 450,000元 | 砍头息的认定：实际借款本金如何确定 |
| 已还款 | 未还款 | 已还100,000元 | 还款事实的认定 |
| 还款性质 | - | 应先抵充利息 | 已还款项抵充本息的顺序 |
| 利息约定 | 年利率12% | 未明确约定 | 口头利息约定的证明标准 |
| 律师费 | 20,000元 | 不予认可 | 律师费是否有合同依据 |

---

## 四、被告抗辩策略

### （一）事实抗辩

1. **借款本金抗辩**：原告存在砍头息行为，实际借款本金应为450,000元，而非500,000元
2. **已还款抗辩**：被告已于2023年9月20日还款100,000元，原告在起诉状中隐瞒此事实

### （二）法律适用抗辩

1. **砍头息法律后果**：根据《民间借贷司法解释》第26条，预先在本金中扣除利息的，应当按照实际出借的金额认定本金
2. **还款抵充顺序**：根据《民法典》第561条，债务人对同一债权人负担数项同种类债务的，除非债务人与债权人约定，否则优先抵充已到期债务；对于利息与本金的抵充顺序，当事人没有约定的，优先抵充利息
3. **律师费承担**：原告要求被告承担律师费没有合同依据，也无法律明确规定

### （三）金额计算

假设认定：
- 实际本金：450,000元
- 已还款：100,000元（2023年9月20日）
- 借款期限：2023年3月15日至2023年9月20日（约6个月）

如果支持利息（假设年利率12%）：
- 6个月利息 = 450,000 × 12% × 6/12 = 27,000元
- 已还款100,000元中，先抵充利息27,000元，剩余73,000元抵充本金
- 剩余本金 = 450,000 - 73,000 = 377,000元

## 五、预期答辩状要点

### 核心答辩观点

1. **借款本金应为450,000元**，而非500,000元（砍头息）
2. **被告已还款100,000元**，应予抵扣
3. **原告主张的律师费无依据**，不应支持
4. **愿意协商分期还款**，体现诚意

### 答辩结构

1. 事实承认与否认
2. 程序性答辩（如需要）
3. 实体性答辩（逐项诉讼请求）
4. 综合请求

## 六、测试要点

### Skill需验证的能力

1. **事实拆解能力**：能否准确识别原告每项事实陈述
2. **争议识别能力**：能否发现原被告事实版本的冲突点
3. **法律适用能力**：能否正确引用砍头息、还款抵充相关法条
4. **格式规范能力**：输出是否符合法定格式
5. **法言法语能力**：是否使用规范法律术语

### 质量评估标准

| 评估项 | 标准 |
|-------|------|
| 诉讼请求拆解 | 每项诉讼请求是否单独列出，无遗漏 |
| 事实分析 | 是否逐条分析原告事实，标明承认/否认 |
| 程序性抗辩 | 是否检查管辖、时效、主体资格 |
| 实体性抗辩 | 是否对每项诉讼请求提出具体抗辩 |
| 法条引用 | 是否引用具体法条（至少到"条"级别） |
| 证据整理 | 是否列出被告方证据及证明目的 |
| 格式规范 | 是否符合法定格式要求 |
| 语言规范 | 是否使用法言法语，无口语化表达 |

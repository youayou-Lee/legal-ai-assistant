3. 保证人责任：连带责任保证的诉讼请求是否单独列明
4. 法条引用：民法典561条是否引用

## 测试案例：民间借贷纠纷（含担保、部分还款）

### 案件基本信息
- **案由**：民间借贷纠纷
- **原告**：张建国（出借人）
- **被告**：李明华（借款人）
- **被告**：王伟（连带责任保证人）

### 借款合同事实
- **借款本金**：800,000元（捌拾万元整）
- **借款时间**：2022-01-01
- **原始借款期限**：1年（至2022-12-31）
- **原始年利率**：15%
- **保证方式**：连带责任保证，保证人王伟

### 补充协议
- **签订时间**：2022-12-20
- **内容**：借款期限延长至2023-12-31，年利率调整为12%

### 还款记录
1. **第一笔还款**：2023-01-20还款100,000元，备注"还本金"
2. **第二笔回款**：2023-04-15还款50,000元，备注"利息"

### 争议焦点
- 剩余本金应如何计算（是否按《民法典》第561条清偿抵充）
- 利息应如何计算（是否分段）
- 保证人责任范围

## 测试关注点

### 1. 当事人适格性
- ✓ 原告是出借人，被告是借款人和保证人
- ✓ 需核查原被告身份信息是否完整

### 2. 管辖法院
- ✓ 被告住所地法院（李明华：北京市朝阳区XX路XX号）
- ✓ 或合同履行地法院（出借人张建国住所地）

### 3. 事实时间线
- 2022-01-01：签订借款合同，出借80万
- 2022-12-20：签订补充协议，延长借期，降利率
- 2023-01-20：还款10万
- 2023-04-15：还款5万
- 2023-12-31：借款到期，被告未还清

### 4. 法律关系认定
- 主合同：民间借贷关系（原告↔被告李明华）
- 从合同：保证合同关系（原告↔被告王伟）
- 保证方式：连带责任保证

### 5. 本金计算（核心测试点）
**v1预期错误行为**：
- 直接从80万本金中扣除10万+5万=15万
- 剩余本金70万元

**v2预期正确行为**（按《民法典》第561条）：
- 第一笔还款10万（2023-01-20）：
  - 当期利息 = 80万 × 12% × 20天/365天 = 5,260.27元
  - 抵充利息后剩余 = 100,000 - 5,260.27 = 94,739.73元
  - 抵充本金后剩余本金 = 800,000 - 94,739.73 = 705,260.27元
- 第二笔还款5万（2023-04-15）：
  - 当期利息 = 705,260.27 × 12% × 85天/365天 = 19,713.34元
  - 5万元不足以抵充全部利息，利息余额 = 19,713.34 - 50,000 = 负数，实际全部抵充利息
  - 剩余本金仍为 705,260.27元
- 实际剩余本金应约**705,260.27元**（v1的70万元是错误的）

### 6. 利息计算
- 第一段：2023-01-20至2023-04-15（85天），本金80万→70.5万
- 第二段：2023-04-15至2023-12-31（260天），本金70.5万

### 7. 证据链完整性
- 借款合同（含担保条款）
- 补充协议
- 银行转账凭证（2022-01-01转账80万）
- 还款记录（两笔）
- 催款通知/律师函（如有）

## 测试案例2：民间借贷纠纷（审查Agent测试）

> 此案例故意包含几个可被审查发现的问题，用于测试多模型审查功能。

### 📋 一键测试

> **用法**：复制下方框内全部内容，粘贴到 OpenClaw 对话框发送即可。

```
帮我写一份民事起诉状，案件情况如下：

【案件基本信息】
案由：民间借贷纠纷
原告：王明华，男，1988年3月15日出生，汉族，住广西壮族自治区南宁市青秀区民族大道168号，联系电话13800138001
被告：李志强，男，1985年7月22日出生，汉族，住广西壮族自治区南宁市西乡塘区大学路100号，联系电话13900139001

【借款合同事实】
借款本金：200,000元
借款时间：2024年6月1日
借款期限：6个月（至2024年12月1日）
年利率：8%

【还款记录】
1. 2024年6月1日，被告向原告支付16,000元（注明为首期利息）
2. 2024年9月1日，被告通过微信转账还款30,000元

【催收情况】
2025年1月10日，原告微信催款，被告回复"过完年一定还"

【诉讼请求】
1. 判令被告偿还借款本金200,000元
2. 判令被告支付自2024年12月1日起至实际清偿之日止的利息（以200,000元为基数，按年利率8%计算）
3. 本案诉讼费用由被告承担

【证据】
1. 《借款协议》
2. 银行转账凭证（2024年6月1日转账20万）
3. 微信转账截图（2024年9月1日还款3万）
4. 微信聊天记录截图
```

**验证重点（审查Agent应发现的问题）**：
1. **砍头息问题**：借款当日（6月1日）被告支付16,000元"首期利息"，实际到手仅184,000元。根据《民法典》第670条，借款本金应按实际出借金额认定，诉讼请求中本金应为184,000元而非200,000元
2. **利息计算基数错误**：如果本金应为184,000元，则利息计算基数也应是184,000元而非200,000元
3. **还款抵充问题**：2024年9月1日还款30,000元，原告主张全部抵充利息，但需按《民法典》第561条清偿抵充顺序处理
4. **管辖法院**：原告住青秀区，被告住西乡塘区，选择西乡塘区法院是否合理（被告住所地管辖，合理）

### 案件基本信息
- **案由**：民间借贷纠纷
- **原告**：王明华（出借人）
- **被告**：李志强（借款人）

### 借款合同事实
- **借款本金**：200,000元
- **借款时间**：2024-06-01
- **借款期限**：6个月（至2024-12-01）
- **年利率**：8%

### 还款记录
1. **2024-06-01**：被告向原告支付16,000元（首期利息）
2. **2024-09-01**：被告通过微信转账还款30,000元

### 事实时间线
- 2024-06-01：签订借款合同，原告转账20万，被告当日支付1.6万"首期利息"
- 2024-09-01：被告微信还款3万，原告抵充为利息
- 2024-12-01：借款到期，被告未偿还剩余本息
- 2025-01-10：原告微信催款，被告回复"过完年一定还"
- 2025-03-15：原告起诉

### 证据链
1. 《借款协议》
2. 银行转账凭证（2024-06-01转账20万）
3. 微信转账截图（2024-09-01还款3万）
4. 微信聊天记录截图

# 法律文书法言法语实务规范研究报告

> 研究日期: 2026-04-08  
> 研究范围: 中国法律文书专业语言规范

---

## 研究发现

### 一、法言法语的核心特征

#### 1.1 用词特征
- **术语精确性**: 使用法定专业术语,如"原告""被告""提起诉讼""主张""辩称"等
- **固定表述**: 
  - "某某诉某某案"
  - "裁定如下""判决如下"
  - "查明认为""认为""鉴于"
- **时间格式**: "20xx年x月x日"而非"xxxx/x/x"
- **金额表述**: 大小写同时注明,如"人民币一万元(¥10,000元)"

#### 1.2 句式特征
- **陈述句为主**: 避免反问、感叹等情感色彩句式
- **长句分层**: 复杂内容分层次表述,确保逻辑清晰
- **主谓宾完整**: 语法结构完整,避免省略关键成分
- **逻辑连贯**: 使用"鉴于""根据""依据""综上"等逻辑连接词

#### 1.3 语气特征
- **客观中立**: 不使用"我觉得""应该"等主观词汇
- **严肃正式**: 符合司法文书的庄重性要求
- **准确严谨**: 避免模糊、歧义表述

---

### 二、法律文书与日常用语的关键区别

| 维度 | 法律文书 | 日常用语 |
|------|---------|---------|
| **用词** | 法定术语,含义固定 | 口语化,多义性强 |
| **句式** | 陈述句,结构完整 | 感叹句、省略句常见 |
| **语气** | 客观、中立、严肃 | 主观、情感化 |
| **格式** | 严格法定格式 | 灵活随意 |
| **逻辑** | 严密,有法律依据 | 情绪化,逻辑松散 |
| **标点** | 仅使用正式标点 | 常用省略号、破折号等 |

### 三、AI优化法律语言的有效方法

根据研究成果,有效方法包括:

#### 3.1 Prompt设计要点
1. **使用法律术语引导**: 
   - 示例: "从合同签订条件、排他性合作和违约责任三个方面分析该合同的履行情况"
   
2. **提供明确诉讼策略**:
   - 明确核心论点
   - 提供优先论证顺序
   - 指定法律依据

3. **分析对方文书模式**:
   - 识别对方诉讼文书的特点
   - 针对性制定反驳策略

4. **迭代优化**:
   - 初稿生成后进行法言法语检查
   - 逐步优化专业表述
   - 确保逻辑连贯性

#### 3.2 优化流程
```
初稿 → 口语化识别 → 术语替换 → 句式调整 → 逻辑优化 → 格式规范
```

### 四、常见的"口语化"错误

#### 4.1 用词错误
- ❌ "我觉得被告应该承担责任"  
  ✅ "被告依法应当承担相应责任"

- ❌ "原告想要10万元赔偿"  
  ✅ "原告请求判令被告赔偿人民币10万元"

- ❌ "这个合同不太公平"  
  ✅ "该合同条款显失公平"

#### 4.2 句式错误
- ❌ 反问句:"难道被告不知道这是违法的吗?"  
  ✅ 陈述句:"被告对其违法性应属明知"

- ❌ 感叹句:"真是太不负责任了!"  
  ✅ "该行为显属严重失职"

#### 4.3 逻辑错误
- ❌ 缺乏法律依据直接陈述观点  
  ✅ "根据《民法典》第XXX条规定..."

- ❌ 逻辑跳跃,缺乏论证过程  
  ✅ "鉴于XXX事实,结合XXX法律条款,XXX结论成立"

#### 4.4 格式错误
- ❌ 使用非正式标点(省略号、破折号等)  
- ❌ 时间格式混乱("2024.1.5""2024/01/05")  
- ❌ 金额仅用小写或仅用大写

---

### 五、法言法语实务要点总结

#### 核心原则
1. **准确性**: 用语精确,避免歧义
2. **规范性**: 符合法定格式和语言规范
3. **严谨性**: 逻辑严密,条理清晰
4. **简洁性**: 言简意赅,避免冗余

#### 实用检查清单
- [ ] 是否使用法定专业术语?
- [ ] 是否存在口语化表述?
- [ ] 句式是否为陈述句为主?
- [ ] 逻辑连接词使用是否恰当?
- [ ] 法律依据是否明确引用?
- [ ] 时间、金额格式是否规范?
- [ ] 标点符号使用是否正式?
- [ ] 是否存在主观情感色彩词汇?

## OpenClaw 配置路径全景

### 核心配置

| 文件 | 用途 |
|------|------|
| `~/.openclaw/openclaw.json` | 主配置：providers、agents列表、auth profiles、channels(飞书)、plugins(tavily/duckduckgo)、gateway端口18789 |
| `~/.openclaw/exec-approvals.json` | 执行审批 |

### 工作空间（Workspace）

| 目录 | 用途 |
|------|------|
| `~/.openclaw/workspace/` | Main Agent 工作空间（调度员） |
| `~/.openclaw/workspace-lawyer/` | Lawyer Agent 工作空间（法律专家） |

每个workspace包含：`AGENTS.md`(核心指令) / `SOUL.md`(身份) / `USER.md`(用户信息) / `TOOLS.md`(工具) / `HEARTBEAT.md`(心跳) / `IDENTITY.md`(身份标识) / `memory/`(记忆)

### Agent 配置

| 目录 | 模型 | 角色 |
|------|------|------|
| `~/.openclaw/agents/main/` | zai/glm-5 | 主调度员，委托法律任务给lawyer |
| `~/.openclaw/agents/lawyer/` | zai/glm-5 | 律师专家，执行法律任务 |
| `~/.openclaw/agents/reviewer-deepseek/` | deepseek/deepseek-chat | 对方律师视角审查 |
| `~/.openclaw/agents/reviewer-qwen/` | openrouter/qwen/qwen3.6-plus:free | 法官视角审查 |

每个agent目录结构：`agent/agent.json` + `agent/models.json` + `agent/auth-profiles.json` + `agent/AGENTS.md` + `sessions/sessions.json`

### Skills（`~/.agents/skills/`）

法律相关：`legal-assistant` / `legal-analysis-quality`(含`legal-share-calc`) / `civil-complaint` / `civil-defense-statement` / `opponent-prediction`

### 其他

- 凭证：`~/.openclaw/credentials/`（飞书pairing等）
- 设备：`~/.openclaw/devices/` + `~/.openclaw/identity/`
- 日志：`~/.openclaw/logs/`
